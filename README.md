# cinema-site-info-observatory
本仓库用于对互联网上已公开的影视类网站进行信息整理与长期观察，重点记录电影与电视剧相关站点的基础公开资料。仓库通过统一结构整理站点名称、访问地址、站点定位说明及基础分类信息，方便进行信息比对与持续维护。本项目不涉及任何影视资源的存储、播放、下载或解析，仅对公开可获取的信息进行整理与记录。所有资料均来源于公开网络环境，整理过程保持客观、中立原则，不对站点内容质量、合法性或服务稳定性作出评价，也不构成任何形式的推荐或使用引导。
